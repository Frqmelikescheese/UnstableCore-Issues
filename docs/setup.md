# Documentation

## Getting Started
1. Install dependencies (Citizens, ProtocolLib).
2. Drop `UnstableCore.jar` into your `/plugins` folder.
3. Restart your server.

## Troubleshooting
- **World Generation Issues?** Delete the `UnderWorld` or `FarLands` folders in the server directory and restart.
- **Dependency Missing?** The plugin will attempt to download them automatically, but for large servers, we recommend installing them manually from the official Spigot/Spigot/GitHub links provided in the `README`.
