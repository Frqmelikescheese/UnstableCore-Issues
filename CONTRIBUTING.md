# Contributing

Thank you for your interest in the Unstable SMP project! 

### Reporting Issues
If you encounter a bug, please use the **GitHub Issues** tab to report it.
*   **Include your server version** and logs from the console.
*   **Be descriptive**: Explain the steps taken before the error occurred.

### Support
* This project is **All Rights Reserved**. Please do not open issues requesting source code or modifications.
* For server-side setup support, refer to the documentation in the `/docs` folder.
